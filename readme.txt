Instructions to run:

Start "IoT project application BETA.py"
Enter http://localhost:5000/ in a browser.
Navigate through the header to view different subpage links.

Layout and funtions
A header with DTU-red color with 3 subpage links.
Enter "/home" for a small greeting.
Enter "/team" for group members and a picture of each.
Enter "/monitoring" to get at random integer between 0 and 100.
